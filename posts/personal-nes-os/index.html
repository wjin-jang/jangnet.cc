Personal/Art/NES OS

November 17, 2025 · Personal, art

Here’s a fun concept I drew:

An operating system for the Nintendo Entertainment System (a.k.a. the Famicom).

You could maybe load games off it?

or use it as a journal with the FAMILY BASIC keyboard?

Nintendo would definitely go for this kind of thing.

I made sure everything fit within 8×8 tiles.

There you have it! A NES Operating System.

Thanks to Inkbox for NESOS, which was the main inspiration for this concept. His is fully functional!
